# history-review
“史流”是一款面向高中历史学考的交互式复习网页，内容依据《中外历史纲要》上下册整理。网页提供中外历史同轴时间线、专题板块和个人学习数据三种模式，收录93个重要历史事件。每个事件包含时间、地点、重要人物、背景、过程、影响、教材页码及空间定位图，支持关键词搜索、时期筛选和掌握标记。系统还会记录学习天数、浏览次数与专题掌握度，帮助学生构建完整的历史知识体系，实现高效复习

使用说明
直接双击 `index.html` 即可使用，无需安装软件，也不需要联网。

## 三种复习方式

同轴时间线：中国历史位于时间轴上方，外国历史位于下方。可以横向拖动、按时期筛选、搜索和缩放。
专题板块：按文明、制度、经济、思想、民族复兴、资本主义、社会主义、国际秩序、全球化九条主线复习。
我的：查看总掌握率、浏览与学习天数、中外及时期分布、专题掌握度和已掌握事件。

点击任一历史事件可以打开完整复习档案，查看重要人物、时间、背景、过程、影响、地点、空间定位图和教材页码，并可标记“已掌握”。复习进度会保存在当前浏览器中。
